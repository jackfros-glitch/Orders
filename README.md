# OrderProcessingApi 

This is a Asp-based backend system for processing customers Orders. It is Built with Scalabilty in mind to handle high traffics spikes. It is also build with Maintainability in Mind as specifiied in the Accessment docs

## Getting Started 

### Clone the repository:

To get started you have to first clone the repository to your local machine

You can do that using ssh 
```bash
git clone git@github.com:jackfros-glitch/Orders.git
```
or https
```bash
git clone https://github.com/jackfros-glitch/Orders.git
```


### Prerequisites

Before you begin, ensure you have the following installed:

    NET SDK (version 8.x)

    MySQL Server (v8+ recommended)




### Installing Dependencies

1. **NET SDK (version 8.x)**

Follow instructions to install the latest version of NET SDK for your platform in the [NET SDK docs](https://dotnet.microsoft.com/en-us/download).

After Installing the NET SDK you can verify by running this command
```bash 
dotnet --version
```
please note for smooth flow with the project version 8.0.0 is recommended

2. **MySQL Server (v8+ recommended)**

Follow instructions to install the latest version of MYSQL Server for your platform in the [MYSQL Server docs](https://www.mysql.com/downloads/).

please note the community version should be more than enought to run the project smoothly.

## SetUp Environment Variable

Before you run the Application locally you would like to setup Your Environment Variables
The Project has been setUp to work with .env files and you can store you Environment Variables
In the .env file for now the only necessary environment variable is the 
```bash
ConnectionStrings__mysqlConnection=Server=localhost;Port=3306;Database=<DBName>;User=<User>;Password=<Password>;
```
The project is setUp to Use MySql so please do so to provide your mysql connection String for a smooth flow 

## SeedDB 

I have set up a Script that you can run the seed the DB so as to enable a fast and smooth in starting up and testing out the project you just need to comment out 
```bash
await app.SeedProducts(); 
```
This is one 103 of the program.cs file to seed the Products Table with sample Data for testing out the 
Application 

## Running the server

From within the root directory

To run the server, execute:
```bash
dotnet run --project Orders.Api
```
The entry point to the aplication is at Orders.Api
